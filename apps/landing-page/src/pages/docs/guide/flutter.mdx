Spooky provides a Dart package for Flutter integration, allowing for natively typed, reactive data bindings.

## Installation

Add `spooky_client` to your `pubspec.yaml`.

```yaml
dependencies:
  spooky_client: ^1.0.0
  flutter:
    sdk: flutter
```
