CLI Reference

Bash 
spky cloud <COMMAND>

Commands:
  login              Authenticate with Sp00ky Cloud (opens browser)
  logout             Clear stored credentials
  create             Create a new cloud project
  list               List cloud projects
  deploy [--upgrade] Deploy current project to the cloud
  status             Show deployment status
  logs [--filter]    Tail logs from cloud deployment
  scale --ssp N      Scale SSP instances
  destroy            Destroy cloud project and all VMs
  backup             Manage database backups
  billing            Billing management
  keys               Manage API keys for CI/CD
  link               Link a GitHub repository for auto-deploy
  env                Manage encrypted environment variables
