Reactive Queries

Subscribe to a SurrealQL query once. sp00ky pushes fresh results the moment the underlying data changes.

sp00ky turns any SurrealQL query into a reactive subscription. Components declare what data they need — sp00ky takes care of keeping it fresh, invalidating the right rows, and pushing updates to every tab, window, and device that cares.

No polling, no manual cache invalidation, no WebSocket boilerplate. Your view layer simply reads from a query result that’s always up to date.

Highlights

  • Full SurrealQL — joins, graph traversals, live aggregates.
  • Sub-50ms reactivity over a single multiplexed WebSocket.
  • Automatic disposal when your component unmounts.
  • Cross-tab invalidation built in.
