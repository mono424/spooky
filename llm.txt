# Sp00ky Sync

> A reactive, local-first sync framework for SurrealDB.

Sp00ky Sync lets you build offline-ready, real-time applications using SurrealDB as both local and remote database. It handles bidirectional sync, conflict resolution, authentication, schema migrations, and file storage — so you can focus on your product.
